# 🌐 Agentic LLM-Powered Web Research System

An intelligent **Agentic AI research assistant** built with **LangGraph** that performs multi-step reasoning through dynamic tool orchestration. The system autonomously selects and invokes tools such as **Wikipedia, arXiv, Tavily Search, and custom web scraping** to gather information, synthesize insights, and generate comprehensive responses.

It features a **stateful graph-based execution pipeline** with conditional tool invocation, **ChatGroq** integration for fast LLM inference, **streaming responses**, and a **Gradio** interface for real-time, multi-turn conversational research.
